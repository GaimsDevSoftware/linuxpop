Plugin Store

Every plugin below ships in the catalogue — click Install inside LinuxPop's Plugin Manager to add it. Missing something? Build your own with the wizard.

Send to Chat AI
AIWeb

Opens your selection in Claude, ChatGPT, Gemini, Perplexity or Google AI — auto-fills the chat box. No API keys, no accounts to wire up.

send_to_ai.py Bundled
Local AI (Ollama)
AIOffline

Explain, summarize, translate, rewrite — runs locally via Ollama. Free, offline, private. Requires Ollama with any model installed.

ollama_ai.py Bundled
Clipboard & Snippets
ClipboardUtility

Rolling history of the last 25 clipboard entries (text and images). Pin and name an entry → it becomes a permanent snippet. Picker via Super+V.

clipboard_history.py Bundled
Word & Character Count
TextUtility

Shows word, character and line counts of the selection as a desktop notification. Useful for tweets, alt-text, abstracts.

wordcount.py Bundled
Case Converter
Text

UPPERCASE, lowercase, Title Case and Slugify — four buttons that transform the selection and put the result on the clipboard.

case_convert.py Bundled
Base64 Encode / Decode
DeveloperEncoding

Encode the selection to Base64, or decode a Base64 string back to text. Two separate buttons. Output goes to the clipboard with a notification.

base64_codec.py Bundled
JSON Pretty-Print
Developer

Reformats unreadable single-line JSON into properly-indented form. Validation errors are shown as notifications, not silent failures.

json_format.py Bundled
URL Encode / Decode
WebEncoding

Percent-encodes text for safe use in URLs, or reverses the encoding. Two buttons for round-tripping.

url_encode.py Bundled
Calculator
UtilityMath

Evaluates math expressions in the selection: 2 + 2, sqrt(16) * sin(pi/2), log10(1000). Result copied to clipboard.

calculator.py Bundled
QR Code
UtilityWeb

Generates a QR code from a selected URL or text — opens the image in your default viewer. Requires qrencode system package.

qr_code.py Bundled
Slugify
TextWeb

Turns "My Post Title!" into "my-post-title". Strips accents, lowercases, replaces non-alphanumerics with hyphens. Result on the clipboard.

case_convert.py Bundled
Open Path
Utility

Selects a file path like ~/Documents/draft.md and clicks "Open" → opens it in the registered application via xdg-open.
