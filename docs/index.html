Install

X11 desktop required. Tested on Linux Mint Cinnamon; works on Ubuntu, Fedora, Arch.

1. System dependencies

# Ubuntu / Linux Mint / Debian
sudo apt-get install -y python3 python3-gi python3-gi-cairo gir1.2-gtk-3.0 \
    gir1.2-handy-1 gir1.2-ayatanaappindicator3-0.1 \
    xclip xdotool xdg-utils python3-xlib

2. Clone & install

git clone https://github.com/GaimsDevSoftware/linuxpop.git ~/linuxpop
cd ~/linuxpop
bash install.sh
python3 ~/linuxpop/main.py    # start now without logging out

3. Use it

  • Highlight any text on screen → popup appears with actions
  • Press Super+Shift+Y → popup on current selection at cursor
  • Press Super+V → clipboard & snippets picker
  • Right-click the tray icon → Settings, Plugins, About
Tip: the first time you open Plugins, browse the catalogue. Most of LinuxPop's power lives in plugins — install only what you'll use.

Three ways to extend it

From zero-code to full-Python.

1 · Install from catalogue

Twelve built-in plugins. Click Install in the Plugin Manager → it's in the popup immediately.

2 · Build with the wizard

No Python required. Open Plugins → Custom → New action. Fill out name, icon, URL or shell template. Done.
